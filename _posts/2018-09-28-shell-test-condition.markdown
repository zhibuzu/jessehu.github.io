## 数值测试

| 参数 | 说明       |
| ---- | ---------- |
| -eq  | 等于则为真 |
| -ne  | 不等       |
| -gt  | 大于       |
| -ge  | 大于等于   |
| -lt  | 小于       |
| -le  | 小于等于   |

### 字符串测试

| 参数      | 说明                   |
| --------- | ---------------------- |
| =         | 等于则为真             |
| !=        | 不想等                 |
| -z 字符串 | 字符串长度为零则为真   |
| -n 字符串 | 字符串长度不为零则为真 |

### 文件测试

| 参数      | 说明                               |
| --------- | ---------------------------------- |
| -e 文件名 | 文件存在则为真                     |
| -r 文件名 | 文件可读则为真                     |
| -w 文件名 | 文件可写则为真                     |
| -x 文件名 | 文件可执行则为真                   |
| -s 文件名 | 文件存在且至少有一个字符则为真     |
| -d 文件名 | 文件存在且为目录则为真             |
| -f 文件名 | 文件存在且为普通文件则为真         |
| -c 文件名 | 文件存在且为字符型特殊文件？则为真 |
| -b 文件名 | 文件存在且为块特殊文件则为真       |
